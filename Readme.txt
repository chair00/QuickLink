# 긴 url을 간결하고 보기쉬운 url로 바꿔주는 프로그램

1. cmd 실행
2. 'python QuickLink.py' 입력
3. 결과로 링크가 표시되는데, 'ctrl+c' 누른채로 마우스로 링크 클릭하면 사이트로 들어가짐
4. 'https://www.yu.ac.kr/pr/yunews/yu-news-room.do?mode=view&articleNo=6467506' 입력창에 입력
5. 제대로 동작될 경우 shorten url 생성 (아닐 경우 invaild 화면 뜸)
6. copy 버튼 누르면 url 복사됨
7. 웹브라우저에 복사된 url 넣고 original site로 이동하는지 확인
